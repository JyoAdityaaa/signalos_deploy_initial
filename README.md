# SignalOS — Convergence-Based Market Intelligence

> **SignalOS** is an AI-driven terminal designed to identify high-probability trading opportunities in the Indian stock market by detecting the "convergence" of independent market signals.

## ⚡ The Pitch

In modern markets, a single news headline or technical breakout is often a "fake-out." True market momentum occurs when multiple independent forces align. **SignalOS** automates the detection of these events. 

By streaming real-time news, institutional bulk deals, and technical patterns into **Gemini 2.0 Flash**, SignalOS filters out the noise to surface **Convergences**—moments where AI sentiment, insider activity, and price action overlap. It’s not just data; it’s high-conviction intelligence.

---

## 🚀 Key Features

### 📊 Intelligence Overview
A high-performance dashboard monitoring the top NSE equities. It tracks active signals and highlights "Convergence Alerts" where the probability of a move is highest.

### 📡 Live Intelligence Feed
A real-time, scrolling stream of raw market data parsed by AI. Every news headline is instantly classified by sentiment and impact, waiting for the convergence engine to evaluate it.

### 🧠 Convergence Engine
A low-latency processing layer that identifies overlaps in a **72-hour window**. It automatically promotes stocks to "Alert" status if they show mixed signals (e.g., a Volume Breakout + Positive AI Sentiment).

### 🌓 Dual-Mode Experience
- **Live Mode**: Connected to real-world NSE data via Yahoo Finance and Google News RSS.
- **Demo Mode**: A sandboxed environment with curated historical "Black Swan" and "Moonshot" events for strategy backtesting and demonstration.

## 🛠️ Technical Architecture

- **Framework**: [Next.js 15](https://nextjs.org/) with App Router.
- **AI Brain**: [Google Gemini 2.0 Flash](https://ai.google.dev/) for structured signal extraction.
- **Real-time Store**: Custom in-memory `LiveStore` singleton for sub-millisecond data propagation.
- **Animations**: [Framer Motion](https://www.framer.com/motion/) for a premium, terminal-like feel.
- **Styling**: Tailwind CSS with a custom "Glassmorphism" design system.

## 📦 Getting Started

### 1. Prerequisites
- Node.js 18+ 
- A Google AI Studio API Key ([Get one here](https://aistudio.google.com/))

### 2. Installation
```bash
git clone https://github.com/your-repo/signalos.git
cd signalos
npm install
```

### 3. Environment Setup
Create a `.env` file in the root directory:
```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) (The app defaults to **LIVE MODE**).
